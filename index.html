Políticas de Privacidade

Caro usuário,

Este documento explica como e onde seus dados são armazenados e como o Conrad garante a segurança dessas informações.

Armazenamento de Dados no Xano

Quando você se cadastra no Conrad, seus dados, como email e senha, são armazenados no Xano, uma plataforma segura de gerenciamento de dados.

Autenticação dos Seus Dados com o Facebook

O Xano integra-se com o Facebook para validar suas credenciais durante o login. Nenhum dado sensível é compartilhado sem sua autorização.

Conexão do Conrad com o Facebook

O Conrad utiliza a API Graph do Facebook por meio da API REST do AppGyver, garantindo comunicação segura e confiável.

Seus Direitos

Você pode acessar, corrigir ou excluir seus dados entrando em contato pelo email claitonwr17@gmail.com.

Segurança dos Dados

Medidas técnicas e organizacionais são adotadas para proteger suas informações contra acesso não autorizado.

Termos de Serviço

Introdução

Bem-vindo ao Conrad! Estes Termos de Serviço regem o uso do aplicativo de análise de postagens Conrad Analítico.

Descrição do Serviço

O Conrad é um aplicativo que analisa postagens em redes sociais para medir engajamento e desempenho.

Uso do Aplicativo

  • Elegibilidade: O uso é permitido apenas para maiores de idade.
  • Conta do Usuário: O usuário é responsável por suas credenciais.
  • Uso Aceitável: É proibido usar o aplicativo para fins ilegais ou prejudiciais.

Análise de Postagens

O Conrad coleta e analisa textos, imagens e vídeos das postagens com o objetivo de oferecer insights sobre o desempenho do conteúdo.
