Políticas de Privacidade do Conrad

Caro usuário,

Este documento explica como e onde seus dados são armazenados e como o Conrad garante a segurança dessas informações.

Armazenamento de Dados no Xano

Quando você se cadastra no Conrad, seus dados, como email e senha, são armazenados no Xano. O Xano é uma plataforma que gerencia e protege dados pessoais. Seus dados são armazenados em uma tabela segura, garantindo que seu email e senha estejam protegidos contra acesso não autorizado. Nem mesmo o criador do aplicativo tem acesso à sua senha ou email.

Autenticação dos Seus Dados com o Facebook

O Xano possui integração com o Facebook. Quando você se cadastra no Conrad, o Xano envia seus dados (como email e senha) para o Facebook, que valida suas credenciais. Através da URL de redirecionamento e da URL da plataforma em que o Conrad foi criado, o Facebook identifica o aplicativo em que você se cadastrou.

Conexão do Conrad com o Facebook

O aplicativo Conrad está conectado ao Facebook via integração com API. Uma API (Interface de Programação de Aplicações) permite que diferentes softwares se comuniquem. No AppGyver, plataforma onde o Conrad foi desenvolvido, há uma API REST que consome dados de outros sites, como o Facebook. Através da API REST do AppGyver, o aplicativo acessa dados do Facebook usando a Graph API do Facebook.

Seus Direitos

Você tem o direito de acessar, corrigir e excluir suas informações pessoais armazenadas pelo Conrad. Para exercer esses direitos, entre em contato conosco através do email claitonwr17@gmail.com.

Segurança dos Dados

Implementamos medidas de segurança técnicas e organizacionais para proteger seus dados contra perda, uso indevido e acesso não autorizado. Continuamos aprimorando nossas práticas de segurança conforme novas tecnologias e padrões se desenvolvem.

Conclusão

Esperamos que este documento tenha esclarecido como o aplicativo Conrad armazena e protege seus dados. Se você tiver qualquer dúvida ou preocupação, por favor, entre em contato conosco.

Termos de Serviço do Aplicativo Conrad

Introdução

Bem-vindo ao Conrad. Este Termo de Serviço rege o uso do aplicativo de análise de postagens fornecido por Conrad Analítico de Postagens. Ao acessar ou usar o Conrad, você concorda em cumprir estes Termos.

Descrição do Serviço

O Conrad é um aplicativo que fornece análise de postagens em plataformas de mídia social. O serviço permite que você faça postagens nas redes sociais, e o aplicativo realiza uma análise de engajamento e desempenho.

Uso do Aplicativo

  • Conta do Usuário: Para acessar certas funcionalidades, pode ser necessário criar uma conta. Você é responsável por manter a confidencialidade de suas credenciais.
  • Uso Aceitável: Você concorda em não usar o aplicativo para fins ilegais, prejudiciais ou que violem os direitos de terceiros.

Análise de Postagens

Dados Coletados: Ao usar o aplicativo, você concorda que podemos coletar e analisar dados de suas postagens, incluindo textos, imagens e vídeos, para que o Conrad possa realizar a análise de seu conteúdo.

Finalidade da Análise: Os dados coletados são usados para fornecer análises e insights sobre o desempenho de suas postagens.
